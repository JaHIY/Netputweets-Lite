What is dabr?

  • A mobile web interface to Twitter's API
  • An open source project by @davidcarrington with inspirations from @whatleydude
  • Secure, storing your Twitter login details in an encrypted cookie on your machine, and never stored on the website

If you have any comments, suggestions or questions then feel free to get in touch.

「奶瓶腿 (NetPutweets)」是什么？

  • 一个中文 Twitter 手机客户端；
  • 一个安全的客户端，不会保存您的登录信息；
  • 基于 @davidcarrington 的 Dabr ＆感谢 @iChada、@17th 的协助修改。
  • 奶瓶腿仍在发育中， @NetPuter 童鞋正努力让他茁壮成长。

如果您有任何建议、意见或疑问，请联系 @NetPuter 童鞋，谢谢！
